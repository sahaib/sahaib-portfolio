# **Serenity Health AI**

![Screenshot of Serenity Health AI App](/og-images/projects/serenity-health-ai.png)

## **Overview**  
Serenity Health AI is a web platform that provides an AI powered companion for mental health. Users can converse in a safe, supportive space and receive guidance on managing stress, anxiety and other concerns. The app focuses on personal growth with features like mood journaling and coping strategy suggestions.  

## **Features**  
- **Supportive chat:** Engage in free form conversations with an empathetic AI.  
- **Focused guidance:** Get targeted advice for specific concerns such as stress, anxiety or productivity.  
- **Mood tracker:** Log emotions with a mood journal to monitor changes over time.  
- **Safety disclaimer:** Clearly informs users that the AI is not a replacement for professional care.  

## **Tech Stack**  
- **Frontend:** React.js, Next.js  
- **AI and API:** OpenAI ChatGPT or similar LLM  
- **Hosting:** Vercel or Netlify for serverless deployment  
- **Utilities:** Tailwind CSS for styling, Zustand for state management  

## **Setup and usage**  
1. Clone the repository and install dependencies with `npm install`.  
2. Add your `OPENAI_API_KEY` in the environment variables.  
3. Start the development server with `npm run dev`.  
4. Open `http://localhost:3000` in your browser.  
5. Interact with Serenity Health AI by selecting your mood and chatting with the assistant.
