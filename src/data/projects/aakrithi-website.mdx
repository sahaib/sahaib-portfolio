# **Aakrithi Website**

![Screenshot of Aakrithi Site](/og-images/projects/aakrithi.png)

## **Overview**  
The Aakrithi website is a **stylized landing page** built to reflect the brand’s essence. It features the company’s products and information in an appealing design.  

## **Features**  
- **Hero Section:** Featured image and tagline.  
- **Product Gallery:** Showcase of products with descriptions.  
- **Responsive Layout:** Optimized for mobile and desktop.  
- **Contact/Form:** An enquiry form for leads.  

## **Tech Stack**  
- **Markup:** HTML5, CSS3  
- **Scripts:** JavaScript (with jQuery for interactivity)  
- **Hosting:** Deployed on a web host (e.g., Netlify or custom domain).  

## **Usage**  
No source repo was provided, but to create a similar site:  
1. Write HTML/CSS based on the design.  
2. Add JavaScript for interactive elements (carousel, form).  
3. Deploy to hosting and connect the domain (aakrithi.co.in).
