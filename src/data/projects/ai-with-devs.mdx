# **AI with Devs**

![Screenshot of AI with Devs Platform](/og-images/projects/ai-with-devs.png)

## **Overview**  
AI with Devs is a learning platform designed to **empower developers with AI tools**. It provides step-by-step project guides, prompts, and resources (like an AI prompt library) so that developers can build real-world applications with modern tech (Next.js, HeroUI, etc.).  

## **Features**  
- **Project Guides:** Step-by-step tutorials for building projects using AI assistance.  
- **Prompt Library:** Curated AI prompts for coding and writing tasks.  
- **Community Resources:** FAQs, documentation, and support for learners.  
- **Modern Tech Stack:** Demonstrates Next.js, HeroUI, and AI integrations in examples.  

## **Tech Stack**  
- **Frontend:** Next.js, React.js (TypeScript)  
- **Styling:** Tailwind CSS, HeroUI components  
- **Backend/API:** Node.js for API routes and integration with AI services (ChatGPT, Cursor IDE, etc.)  
- **AI:** Uses OpenAI API and other AI tool integrations for learning features.  

## **Getting Started**  
1. Clone: `git clone https://github.com/sahaib/aiwithdevs.git` (if available) or fork the template.  
2. Install: `npm install`.  
3. Configure: Add any required API keys (e.g. OpenAI) in `.env.local`.  
4. Run: `npm run dev` to launch locally at `localhost:3000`.
